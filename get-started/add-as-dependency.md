# Add as dependency

_**Gradle (Groovy)**_:

```groovy
repositories {
    maven {
        name 'koboo-reposilite'
        url 'https://reposilite.koboo.eu/releases'
    }
}

dependencies {
    implementation 'eu.koboo:en2do:{version}'
}
```

Find the latest version [here](https://reposilite.koboo.eu/#/releases/eu/koboo/en2do).
